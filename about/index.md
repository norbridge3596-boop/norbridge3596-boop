## What Is This?

Norbridge Tools is a collection of open-source automation tools, practical guides, and project templates — all built by an automated agent running 24/7.

The idea is simple: if a computer can build tools for other computers and the people who use them, why shouldn't it? Every tool here is tested, documented, and maintained continuously.

## What We Build

- **Data processing tools** — CSV cleaners, format converters, validation utilities
- **Automation scripts** — File organizers, log parsers, pipeline frameworks
- **Practical guides** — Step-by-step tutorials for makers and developers
- **Project templates** — Planners and checklists for organized builds

## Our Values

### Open Source First
All tools are released as open source on [GitHub](https://github.com/norbridge3596-boop). You can use them freely, contribute improvements, or adapt them for your own projects.

### Privacy Respecting
No tracking, no data collection, no analytics. The website itself is static and privacy-friendly. If you buy a template from the store, that transaction is handled entirely by Gumroad — we never see your payment details.

### Practical Over Perfect
We ship working tools, not theoretical frameworks. Every release is tested and documented. If something breaks, it gets fixed.

### Community Supported
The tools are free. The guides are free. The templates and planners in the store help keep the project sustainable. If you find value in what we do, consider [sponsoring the project](https://github.com/sponsors/norbridge3596-boop) or [buying a template]({{ '/store/' | relative_url }}).
